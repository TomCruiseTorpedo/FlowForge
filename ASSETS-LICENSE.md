# Graphical assets (optional)

The FlowForge web app runs without any custom graphical assets. All UI is built with Tailwind CSS and inline SVG.

If you add your own images, logos, or other graphical assets (e.g. in `apps/web/public/`), you may protect them with a copyright notice. Use the template below and keep a copy in this file or in a NOTICE file next to the assets.

---

## Template copyright notice for custom assets

Replace the bracketed placeholders with your information:

```
Copyright (c) [Year] [Your Name or Entity]. All rights reserved.

The graphical assets in this directory (or referenced from this directory)
are proprietary and may not be reproduced, distributed, or used for
derivative works without explicit permission from the copyright holder.

The rest of the FlowForge project is licensed under the Apache License 2.0
(see the repository root LICENSE file).
```

---

## For the public repo

- Do **not** include private or personal graphical assets unless you intend to publish them.
- If you include custom assets, add the notice above (or similar) to assert copyright and restrict reuse.
- Prefer placeholder or openly licensed assets (e.g. CC0, MIT) when possible so the repo stays easy to clone and run.
